       mirrors what DeviceCapabilityManager does and is the correct probe.

       import javax.annotation.Nullable — write @javax.annotation.Nullable inline to avoid import collision with
       other @Nullable annotations). Kotlin ? alone is not sufficient — KSP needs the Java annotation to generate
       null-safe Dagger code.
