| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-32-10 | Spoofing | GenAiClient nullable injection | accept | Null client is the safe default — AI features simply unavailable. Phase 34 AiModule controls when a real client is provided (AIFND-11) |
