## Component: ScrollToTopFab

A floating action button that appears when users scroll down past a threshold, allowing them to quickly return to the top of any scrollable list.

### Visual Specification

| Property | Value |
|----------|-------|
| Icon | `Icons.Default.KeyboardArrowUp` |
| Icon size | 24dp |
| Button size | 48dp × 48dp |
| Corner radius | 28dp (circular) |
| Background | `MaterialTheme.colorScheme.secondaryContainer` |
| Icon tint | `MaterialTheme.colorScheme.onSecondaryContainer` |
| Elevation | 6dp |

### Positioning

| Property | Value |
|----------|-------|
| Horizontal position | Center (50% from left) |
| Vertical position | 16dp from bottom edge |
| Anchor | Parent container bottom |

### Animation

| Property | Value |
|----------|-------|
| Enter | Fade in with `fadeIn()` (300ms, ease-out) |
| Exit | Fade out with `fadeOut()` (200ms, ease-in) |
| Scroll target | Animate scroll to index 0 with `animateScrollToItem(0)` |

---

## Visibility Contract

| State | Trigger | Result |
|-------|---------|--------|
| Hidden | `firstVisibleItemIndex == 0 && firstVisibleItemScrollOffset <= 50` | Button invisible |
| Visible | `firstVisibleItemIndex > 0 \|\| firstVisibleItemScrollOffset > 50` | Button fades in |

### Threshold

| Token | Value | Usage |
|-------|-------|-------|
| scrollThreshold | 50dp | Offset to trigger visibility |

## Modifier API Contract

```kotlin
@Composable
fun Modifier.scrollToTop(
    lazyListState: LazyListState,
    threshold: Dp = 50.dp,
    bottomMargin: Dp = 16.dp
): Modifier
```

**Behavior:**
- Automatically manages visibility based on scroll position
- Animates fade in/out
- On click: scrolls to item index 0 with animation
- Works with any LazyColumn via state injection
