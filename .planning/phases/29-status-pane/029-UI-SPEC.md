---

## Visuals

### Focal Point
StatusPaneHero (net position hero number) — largest, most prominent element on screen, colored per Positive/Negative Indicator rules.

### Visual Hierarchy (Priority Order)
1. **Primary:** StatusPaneHero (net position + month/year label + currency symbol)
2. **Secondary:** StatusPaneFigureGrid (2-column grid of 6 supporting figures: Net Cash Flow, Income, Expense, Savings, Lending, Borrowing)
3. **Tertiary:** Empty state (StatusPaneEmptyState) or error state (StatusPaneErrorState) when no data available

### Layout Rules
- Hero centered horizontally at top of scrollable column
- Figure grid uses 2-column arrangement below hero with 16dp spacing
- Empty/error states fill entire screen, centered vertically and horizontally
